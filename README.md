# Nexiora Technologies — Landing Page

Pre-launch waitlist landing page for Nexiora Technologies (data analysis & Power BI dashboard service).

## Files
- `index.html` — the full landing page
- `waitlist-backend.gs` — Google Apps Script backend for the waitlist (Sheet storage + optional Resend confirmation email)
- `SETUP_INSTRUCTIONS.md` — step-by-step setup guide for the backend

See `SETUP_INSTRUCTIONS.md` to connect the waitlist form to a live backend before deploying `index.html` on a custom domain.
